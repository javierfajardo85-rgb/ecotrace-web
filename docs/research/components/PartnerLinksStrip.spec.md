# PartnerLinksStrip Specification (Solugen-inspired)

## Overview
- **Target file:** `src/components/PartnerLinksStrip.tsx`
- **Placement:** Home page (`src/app/page.tsx`), immediately after `PartnerWithUsBanner`

- Full-bleed black background section (image later).

- Centered row of 4 pill buttons with circular image slots (placeholders).
